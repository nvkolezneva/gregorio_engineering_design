# Интернет - магазин

### Как запустить?
`cd client`
=
`npm i`
=
`ng serve`
=
`cd ..`
=
`cd server`
=
`json-server --watch db.json`
=
##Если не установлен ангуляр, то нужно установить его глобально командой:
`npm install -g @angular/cli`
=
